Project 02

COA 国服控制台

这是一个独立的控制台页面，用于向指定 Gist 发送固定动作，并读取 COA 国服的运行状态。页面只保存本地连接配置，不会把 Token 或 Gist ID 写回远端。

  • 通过 GitHub Gist API 读写 `command.json` 与 `status.json`。
  • 支持本地缓存 Token、Gist ID 和启动账号别名。
  • 保留刷新冷却、操作冷却和状态摘要展示。
